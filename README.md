# Shwe Sin Shopkeeper

A simple mobile-first shopkeeper web app for adding items, storing their photo, name, amount, and price, and searching items quickly in English or Burmese.

## Features
- Add items with a photo, name, amount, and price
- Search items by typing the name
- Toggle between English and Burmese
- Clean responsive layout for Android screens
- Local storage based, so it works without a backend

## Run locally

```bash
npm install
npm run dev
```

## Build for Vercel

```bash
npm run build
```

Deploy the project to Vercel as a standard Next.js app.
